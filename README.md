# 🤝 KeenKeeper

## 📖 Description
KeenKeeper is a comprehensive web application designed to help users maintain meaningful relationships. It allows you to track interactions, monitor contact frequency, and manage your social network through a clean, responsive, and intuitive interface.


## 🚀 Technologies Used
* **Frontend :** React.js, Next.js
* **Styling :** Tailwind CSS, DaisyUI
* **Icons :** React Icons
* **Data Persistence :** LocalStorage API
* **Deployment :** Vercel

## ✨ Key Features
1. **Interactive Dashboard :** Quickly view your total connections, on-track relationships, and friends needing attention through a real-time statistical grid.
2. **Dynamic Timeline :** Track all your interactions (Call, Text, Video) with a sortable timeline that keeps you updated on your communication history.
3. **Smart Status System :** Automated status indicators (On-Track, Almost Due, Overdue) to ensure you never lose touch with the people who matter most.
